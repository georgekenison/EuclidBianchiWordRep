# EuclidBianchiWordRep
Sage code related to the Word Representation for Euclidean Bianchi Group elements.  The sage code in the files relates to the exhaustive search for matrices in each of the Euclidean Bianchi Groups.

The repository contains python notebooks (run using a SageMath kernel) for each of the Euclidean Bianchi groups
- PSL-d1.ipynb
- PSL-d2.ipynb
- PSL-d3.ipynb
- PSL-d7.ipynb
- PSL-d11.ipynb

The repository also contains a brief explanation for the raw code.
- sage-code.tex
- sage-code.pdf
- splncs04.bst
- preamble.tex
